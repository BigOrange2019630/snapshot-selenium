Installation
================================================================================


You can install snapshot-selenium via pip:

.. code-block:: bash

    $ pip install snapshot-selenium


or clone it and install it:

.. code-block:: bash

    $ git clone https://github.com/pyecharts/snapshot-selenium.git
    $ cd snapshot-selenium
    $ python setup.py install
